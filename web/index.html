Proyecto de Análisis de Datos de Airbnb Málaga.


Descripción

Realicé un proyecto de análisis de datos utilizando Python, Tableau y PostgreSQL para explorar el conjunto de datos de Airbnb en Málaga.
Mi objetivo ha sido aprender más acerca del Análisis de Datos y obtener resultados sobre la oferta de alojamiento en la ciudad y las tendencias del mercado.
